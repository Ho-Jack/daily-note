## 立即执行函数的几种写法
```js
!function(){

}()
//！号，可以替换成 ~ , * , + , - , ! 等其他一元操作符，
//如果没有这个加号的话，解析器会认为function是一个函数声明的开始，而后面（）将会导致语法错误。
//在function前面加上+号时，就变成了一个函数表达式，而函数表达式后面又添加了一个()就变成了一个立即执行的函数了。
```

```js
(function() {
      console.log('Foo!'); 
})();
```

```js
(function() {
     console.log('Foo!'); 
}());
```

###  作用: .创建一个独立的作用域，避免全面局污染

 不破坏污染全局的命名空间，若需要使用，将其用变量传入如

```js
 (function(window){                
   console.log('Foo!'); 
 }(window))
```
