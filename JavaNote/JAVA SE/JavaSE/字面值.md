# 字面值

在java代码中我们可以用一些值来代表一些数据。这些值就是字面值。

Java还对这些字值进行了一定的分类。

## 字面值分类

- 整数值    例如： 10   99  66 
- 小数值    例如：  3.14   5.78
- 字符串值   用英文双引号包裹起来   例如:    "三更草堂"
- 布尔值    用来表示真或假  只有两种值  true和false
- 空值        代表空值，我们后面会使用到    null
